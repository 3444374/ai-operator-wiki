# 知识库同步规则

> 项目是知识唯一来源，知识库是编译查询界面。
> 项目内 agent 操作指南：`research/knowledge_sync_guide.md`

## 同步命令

```bash
cd ../ai-operator-wiki && bash sync-wiki.sh
```

同步后：打开 Obsidian → `Cmd+P` → "Ingest from folder" → `raw/`

## 自动同步（通配符，无需配置）

- `research/*.md` → `raw/references/`
- `reading_notes/*.md` → `raw/papers/`
- `reference/*.pdf` → `raw/papers/`
- `experiments/plans/*.md` → `experiments/plans/`

## 新增知识目录

1. 在 `sync-wiki.sh` 加 `cp` 行
2. 在项目 `PROJECT_LOG.md` 记录
3. `git commit` 脚本

## 不同步

代码、CSV、规则文件、沟通记录
