> **模板说明**：本模板分 **4 层**，从浅到深。第一层是快速索引，第二层分析论文本身，第三层做批判性判断，第四层连接你的课题。

> ⏱ **预期耗时**：第 1 层 ~10 分钟 → +第 2 层 ~30 分钟 → +第 3 层 ~20 分钟 → +第 4 层 ~20 分钟。**全部填完约 80 分钟**，这才是精读的真实成本。

> 并非每个板块都必须填——但高层分析（第三、四层）才是精读的增值所在。如果一篇论文你只填了第一、二层，说明它不值得精读，标记为泛读即可。

## ▎第三层 · 批判性评估

**目的**：不盲目相信论文的叙述，做研究者的独立判断。

### 1. 假设检验

论文中有哪些**没有明说但实际依赖的假设**？在什么条件下这些假设不成立？

- 假设 1：（例如：所有模型调用成本相同）
  - 反例 / 边界：
- 假设 2：
  - 反例 / 边界：

### 2. 边界探查

- **方法适用边界**：论文方法在什么场景下不再有效？
- **扩展性限制**：数据量再大 10× / 100× 会怎样？模型调用再密集呢？
- **复现难度**：代码开源？依赖专有组件？实验可复现？

### 3. 可信度评估

| 维度 | 评价 | 依据 |
|------|------|------|
| 实验公平性 | 🟢 公平 / 🟡 有疑点 / 🔴 不利 | |
| 结果显著性 | 🟢 显著 / 🟡 勉强 / 🔴 不显著 | |
| 开源/可复现 | 🟢 全开 / 🟡 部分 / 🔴 闭源 | |
| 论文自身局限 | 是否诚实地讨论了局限？ | |

### 4. 与同行工作的对比

（这篇论文相比同方向 SOTA 的新贡献在哪？）

- 比 **[Paper A]** 多了什么？
- 比 **[Paper B]** 少了什么 / 假设了什么不同？
- 在 **[你的课题]** 的坐标系中，它属于什么位置？（DB4AI / 外部执行 / ...）
