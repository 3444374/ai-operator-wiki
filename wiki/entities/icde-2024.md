# ICDE 2024

## 描述
ICDE 2024（International Conference on Data Engineering）是数据库与数据工程领域的国际顶级学术会议，被中国计算机学会（CCF）推荐为 A 类会议。该届会议收录了由[[entities/huawei|华为]]与[[entities/tsinghua-university|清华大学]]研究团队合作完成的论文《GaussML: An End-to-End In-database Machine Learning System》（作者 Li、Sun 等），该论文详细阐述了[[entities/gaussml|GaussML]]系统并提出了[[concepts/ai-sql-operators|AI SQL 算子]]等关键技术。ICDE 会议严格的同行评审流程保障了所收录成果的学术可信度，因此本知识库将其列为来源级别标识，以强化 GaussML 方案的权威性基础。此外，本知识库于 2026-07-15 对上述论文进行了系统性的阅读与分析，使得 ICDE 2024 同时成为文献引用的时间锚点。
