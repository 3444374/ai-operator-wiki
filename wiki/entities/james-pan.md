# James Pan

## 描述
James Pan 是 [[entities/清华大学|清华大学]] 计算机科学与技术系的研究人员，与 [[entities/guoliang-li|Guoliang Li]] 密切合作，共同在 [[entities/vldb-2025|PVLDB 2025]] 上发表了题为“Database Perspective on LLM Inference Systems”的 tutorial 论文。该工作系统化地将 [[concepts/llm-inference-systems|LLM Inference Systems]] 的技术栈与数据库核心概念进行类比，为数据库领域研究者提供了一个快速理解推理系统设计空间的框架，其中重点阐释了 [[concepts/four-layer-inference-stack|四层推理技术栈]]。他的研究聚焦于云数据库、数据库与人工智能的交叉领域，致力于将数据库领域的优化思想引入大语言模型推理系统。
