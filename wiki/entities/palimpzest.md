# Palimpzest

## 描述
Palimpzest 是一个面向检索增强生成（RAG）场景的语义查询系统，能够通过声明式接口将自然语言查询转换为高效的数据库操作。在 Galois 系统的实验评估中，Palimpzest 被用作对比基线之一。实验结果表明，[[entities/galois|Galois]] 在同一 RAG 任务（MC 数据集）上以仅约 1/11 的 token 成本取得了与 Palimpzest 接近的答案质量，验证了 Galois 在成本控制与效能优化方面的优势。该对比进一步揭示了检索增强生成系统中成本‑质量权衡的重要性。
